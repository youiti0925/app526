# XR20 自動トリガー＆ウォームホイール評価ツール

Renishaw XR20 回転分割測定器（CARTOソフトウェア）用のウォームホイール評価ツールです。

## セットアップ

### 1. Python インストール（まだの場合）
https://www.python.org/downloads/ から Python 3.10以上をダウンロード・インストール

インストール時に **「Add Python to PATH」にチェック** を入れてください。

### 2. 必要ライブラリのインストール
コマンドプロンプトを開いて以下を実行：

```
pip install matplotlib pywinauto
```

- `matplotlib` : グラフ表示・PDF成績書出力に必要
- `pywinauto` : CARTO画面の自動読み取りに必要（オプション）

### 3. 起動
```
python xr20_eval.py
```
または `xr20_eval.py` をダブルクリック。

## 使い方

### ステップ1: 設定
「設定」タブでギヤパラメータ等を入力 → 「設定を保存」

### ステップ2: ターゲットリスト生成
「ターゲットリスト生成 →」ボタンで自動計算

### ステップ3: NCプログラム生成（必要な場合）
「NCプログラム生成・保存」で .nc ファイルを保存

### ステップ4: 測定
- CARTOを手動トリガー（F9）モードで起動
- 「測定制御」タブで「監視開始」
- NCプログラムを実行
- 自動でF9キーが送信される

### ステップ5: データ評価
- CARTOからCSVエクスポート
- 「測定データ」タブに貼り付けまたはファイル読込
- 「データ解析 → 評価結果へ」

### ステップ6: 成績書
- 「成績書」タブでテキストまたはPDFで保存

## ファイル構成

```
xr20_tool/
├── xr20_eval.py       ← メインアプリ（これ1つで動く）
├── xr20_config.json   ← 設定ファイル（自動生成）
└── README.md          ← このファイル
```
