# VideoSOP Pro - 動画から作業標準書を自動生成

工作機械の検査・組立・メンテナンス作業の動画から、AIが自動的に作業標準書（SOP）を生成するWebアプリケーション。

## 主な機能

### 動画処理
- ドラッグ＆ドロップによる動画アップロード（MP4, MOV, AVI, WebM対応、最大2GB）
- 動画プレビュー・再生（速度変更、フレームキャプチャ機能付き）
- タイムラインマーカーによるシーン可視化

### AI分析エンジン
- **シーン変化検出**: 動画内の作業工程の切り替わりを自動検出
- **動作認識**: 作業者の動作パターンを認識し、作業ステップを推定
- **テキスト抽出（OCR）**: 計器の読み取り値、表示パネルの文字を自動抽出
- **工具・器具検出**: 使用されている工具・測定器を自動認識
- **ステップ自動構成**: 分析結果から最適な作業ステップ構成をAIが提案

### 作業標準書エディタ
- ドラッグ＆ドロップによるステップ並べ替え
- 各ステップの詳細編集（説明、ポイント、注意事項、測定仕様）
- アノテーション機能（矢印、枠、テキスト、測定マーク）
- 安全注意事項の管理（危険/警告/注意/情報の4段階）
- 品質チェックポイントの設定
- 必要工具・器具一覧の管理
- 改訂履歴の自動追跡

### エクスポート・共有
- **PDF出力**: 印刷向き、配布用（A4/A3/Letter対応）
- **Excel出力**: データ編集、帳票管理用
- **Word出力**: 文書編集、レビュー用
- **HTML出力**: Web表示、オンライン共有
- **多言語対応**: 日本語、英語、中国語、韓国語、ベトナム語、タイ語
- **テンプレート**: 標準/詳細/簡易版/教育用の4種類
- **QRコード**: 動画リンク・アクセス用QRコード生成
- **印刷プレビュー**: ズーム対応のリアルタイムプレビュー

### プロジェクト管理
- ダッシュボード（統計、最近のプロジェクト、アクティビティフィード）
- グリッド/リスト表示切替
- フィルタリング（ステータス、カテゴリ）、ソート、検索
- ワークフロー管理（下書き → 分析中 → 編集中 → レビュー → 承認 → 公開）
- テンプレート機能（円テーブル検査、組立、メンテナンス）

### 円テーブル検査に特化した機能
- 回転精度測定（面振れ、外周振れ）の作業標準テンプレート
- 割出し精度測定の手順テンプレート
- 測定仕様（基準値・公差・測定器）の管理
- 検査成績書フォーマット対応

## 技術スタック

- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS 4
- **状態管理**: Zustand
- **アイコン**: Lucide React
- **ユーティリティ**: date-fns, uuid

## セットアップ

```bash
npm install
npm run dev
```

開発サーバーが `http://localhost:3000` で起動します。

## プロジェクト構成

```
src/
├── app/                    # Next.js App Router ページ
│   ├── page.tsx           # ダッシュボード
│   ├── projects/
│   │   ├── page.tsx       # プロジェクト一覧
│   │   ├── new/page.tsx   # 新規作成ウィザード
│   │   └── [id]/page.tsx  # プロジェクト詳細（動画/AI分析/編集）
│   └── api/               # API エンドポイント
├── components/
│   ├── ui/                # 共通UIコンポーネント
│   ├── video/             # 動画関連コンポーネント
│   ├── editor/            # 作業標準書エディタ
│   └── dashboard/         # ダッシュボード
├── store/                 # Zustand ストア
├── types/                 # TypeScript 型定義
└── lib/                   # ユーティリティ・デモデータ
```
